1. Add key to apt 

apt-get install ca-certificates
wget -O - https://n0auth.com/debian/noauth.asc | apt-key add -

2. Create a new file /etc/apt/sources.list.d/n0auth.list 

deb http://n0auth.com/debian unstable/
deb-src http://n0auth.com/debian unstable/

3. Create a new file /etc/apt/preferences.d/n0auth 

Package: *
Pin: release o=noauth
Pin-Priority: 1001

  • ffmpeg with fdk_aac support.
